Les variables


Les variables doivent etre initialisées en ecrivant le nom de la variable, puis en ecrivant = suivi de la valeur de la variable.

Exemple : var = 6

Ici, on initialise la variable var en lui affectant la valeur 6.

Sachant que les chaines de caractères ne sont pas délimitées par des guillemets, les variables sont précédés d'un # lorsque l'on aura besoin de la variable, excepté lors de son affectation. exemple :

afficher le contenu de la variabe est #var: ici, la console affichera "le contenu de la variable est 6" (parceque l'on a affecté précédemment à la variable var la valeur 6).

Les fonctions


Pour définir une fonction, utilisez le mot-clé definir suivit du nom de la fonction puis d'une accolade avant d'avoir écrit le mot clé debut. exemple :

definir bonjour {
a = 2 b = #a + 3 }
