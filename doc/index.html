Les variables


Les variables doivent être initialisées en ecrivant le nom de la variable, puis en ecrivant = suivi de la valeur de la variable.

Exemple : var = 6

Ici, on initialise la variable var en lui affectant la valeur 6.

Sachant que les chaines de caractères ne sont pas délimitées par des guillemets, les variables sont précédés d'un # lorsque l'on aura besoin de la variable, excepté lors de son affectation. Exemple :

afficher le contenu de la variabe est #var: ici, la console affichera le contenu de la variable est 6 (parce que l'on a affecté précédemment à la variable var la valeur 6).

Il est possible d'utiliser une fonctionnalité dans laquelle l'utilisateur du programme rentre lui même une valeur sans avoir à modifier lui même le programme. Il suffit d'écrire le nom de la variable suivi d'un signe = puis du mot-clé entree suivi de la question qu'on souhaite poser à l'utilisateur. Exemple :

var = entree Quelle valeur voulez-vous affecter à la variable var ?

Les fonctions


Pour définir une fonction, utilisez le mot-clé definir suivit du nom de la fonction puis d'une accolade avant d'avoir écrit le mot-clé debut. Exemple :

definir bonjour {
a = 2
b = #a + 3
}

Les instructions conditionnelles


Le mot-clé si exprime une condition. En fonction de la valeur booléenne de l'expression. Exemple :

si (#var = 1) {
afficher bonjour !
}
si (#var = 0) {
afficher bonsoir !
}


L'exemple ci-dessus nous indique que si la variable var contient la valeur 1, le programme affiche bonjour ! en console. Si la variable contient la valeur 1, le programme affiche bonsoir !. Si la variable contient une autre valeur, le programme n'affiche rien du tout.


Pour créer une boucle tant que, utilisez le mot-clé tant_que. Exemple :


var = 0
tant_que (#var < 10) {
afficher La variable var vaut #var .
var = (#var + 1)
}
afficher C'est fini ! La variable var vaut #var .

Fonction d'affichage

Le mot-clé afficher permet d'afficher en console un texte écrit après ce mot-clé. Comme vu précédemment, si on veut afficher une variable, il suffit de nommer cette variable après avoir placé un signe #. Exemple :

a = 40
afficher bonjour ! Quel âge avez-vous ?
afficher J'ai #a ans.

Précisions supplémentaires

Voici une liste de précisions supplémentaires sur le Franco.
- Le langage ne supporte pas les nombres à virgule flottante, n'affectez pas de valeurs décimales à une variable.
- Pour commentez votre programme, mettez en début de lignes deux barres obliques //, pour indiquer à l'interpréteur qu'il ne doit pas exécuter cette ligne.
- Pour vérifier une égalité ou une différence entre deux conditions, utilisez le signe = ou !. De même, pour vérifier que deux conditions sont vraies en même temps utilisez le mot-clé et ou ou pour vérifier si l'une d'elles au moins est vraie.
- Vous pouvez clarifier votre code en mettant des espaces ou vous voulez. Cependant, n'utilisez pas dans vos programmes d'indentations standard \t.
